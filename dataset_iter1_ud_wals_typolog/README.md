## Summary

Compiled typological metadata for 186 UD languages joining WALS features (81A word order, 49A case count) with UD-derived case richness via ISO 639 codes. Contains 3 datasets: full_typological_profiles (186 languages, all fields), word_order_prediction (80 languages with WALS 81A labels as classification targets), and ud_case_richness (81 languages with observed morphological case values). Sources: WALS CLDF GitHub CSVs + HuggingFace UD treebanks (commul/universal_dependencies, streaming 5k sentences per treebank).
