## Summary

5.7M per-dependency records extracted from 20 typologically diverse Universal Dependencies treebanks (commul/universal_dependencies on HuggingFace). Each record encodes: treebank_id, dependency relation, dependent/head POS tags, sentence length, morphological Case features, and dependency distance as output. Covers case-rich languages (Finnish 15 cases, Hungarian 22, Basque 15, Turkish 8, Czech 8, Russian 8, Hindi 7) and analytic languages (English, Chinese, Japanese, Korean). Includes treebank-level metadata (sentence stats, case inventories) and sentence-level MDD observed/random baselines. Split into 19 files under 100MB each.
