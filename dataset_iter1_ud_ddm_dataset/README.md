## Summary

Pre-aggregated dataset from 314 Universal Dependencies treebanks (173 languages) with per-treebank DDM statistics (naive and sentence-length-standardized), DDM(n) by sentence length with permutation-sampled random baselines, WALS 81A word order classifications (65% coverage), case richness from UD morphological features, and 500k stratified-sampled core-argument dependency records (nsubj/obj/iobj) with distance, case marking, and typological covariates for Cox proportional hazards modeling. Two datasets: ud_treebank_summaries (314 treebank-level summaries) and ud_core_argument_deps (500k individual dependency records).
